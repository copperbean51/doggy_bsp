
# Manual installation

Manual installation is an advanced method. 

### Install this repository 

```sh
sudo apt update
sudo apt upgrade
reboot
```

```
sudo apt install -y git python3-pip
reboot
```

```
cd ~
 git clone https://github.com/copperbean51/doggy_bsp.git
cd doggy_bsp
./install.sh	
sudo reboot
```

cd ~/doggy_bsp
